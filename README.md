# Movies Dashboard Analysis

## Project Overview
This project focuses on analyzing movie ratings, genres, and audience preferences using Power BI dashboard visualizations. The goal of the project is to identify trends in movie performance, genre popularity, and rating distribution to generate meaningful business insights.

---

## Objective
The objective of this project is to analyze movie-related data using Power BI dashboards and identify audience preferences, rating trends, and genre performance through data visualization and storytelling techniques.

---

## Dataset Overview
The dataset contains movie-related information including:

- Movie Titles
- Genres
- Ratings
- Release Years
- User Reviews
- Rating Distribution
- Genre Performance

The project was developed using Power BI dashboard visualization and data analysis techniques.

---

The following analysis was performed:

## Key Business Insights
- Comedy genre contained the highest number of movies.
- Film-Noir and Mystery genres showed higher average ratings.
- Most movies received ratings within the medium to high range.
- Audience preferences varied across genres.
- Certain genres consistently performed better in user ratings.

---

## Statistical Validation

### Hypothesis Testing

### Null Hypothesis (H₀)
Average movie ratings are equal across all genres.

### Alternative Hypothesis (H₁)
Some movie genres have significantly different average ratings.

A basic statistical comparison was performed to evaluate rating differences across genres. The findings suggest that audience ratings vary between genres, indicating different levels of audience preference.
