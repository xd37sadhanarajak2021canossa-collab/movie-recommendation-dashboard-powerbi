# Movie Analytics Dashboard (Power BI)

## Objective
To analyze movie ratings data and build an interactive dashboard to derive insights about user behavior and movie performance.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Dataset
- Movies
- Ratings
- Tags

## KPIs
- Average Rating
- Total Movies
- Total Ratings
- Total Users

## Key Insights
- Most users give ratings between 3 and 5
- Drama and Comedy genres have higher average ratings
- Ratings activity decreased over time
- A small number of users contribute most ratings

## Dashboard Features
- Genre-wise analysis
- Rating distribution
- Time-based trends
- Interactive filtering
